마운트된 팀 컨텍스트를 로컬에서 제거한다.

## 실행 절차

1. `~/.claude/CLAUDE.md`에서 `<!-- tc:` 로 시작하는 블록 찾기
   - 없으면: "마운트된 팀 컨텍스트가 없습니다." 출력 후 종료
   - 여러 개면: 목록 보여주고 사용자에게 선택 요청
2. `<!-- tc:{id} -->` ~ `<!-- /tc:{id} -->` 블록 전체 삭제
3. `~/.claude/team-context/{id}/` 디렉토리 삭제
4. 출력: `✅ {name} 팀 컨텍스트가 제거되었습니다.`
