문서 지도 기반으로 필요한 팀 문서만 on-demand fetch한다.

## 실행 절차

1. `~/.claude/team-context/{id}/README.md`의 문서 지도 테이블 읽기
2. 사용자 질문과 관련된 경로 식별
3. `{team_url}/{path}`로 해당 문서만 fetch
4. 답변에 활용

## 규칙

- 매번 전체를 fetch하지 않는다. 필요한 것만 on-demand.
- 문서 지도에 없는 경로는 fetch 시도하지 않는다.
- fetch 실패 시 사용자에게 알리고 대안 제시.
- 모르는 용어를 만나면 `glossary/` 에서 먼저 검색.
