## 시작하기

1. 이 레포를 fork하세요
2. `README.md`의 frontmatter를 팀에 맞게 수정하세요
3. 팀 문서를 원하는 구조로 추가하세요
4. Claude Code에서 마운트하세요:
   ```
   이 URL을 보고 팀 컨텍스트를 설치해줘: https://[your-pages-url]/SETUP.md
   ```
